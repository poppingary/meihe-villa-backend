# 梅鶴山莊文化資產 (Meihe Villa Cultural Heritage)

## 文化資產登錄資訊

| 項目 | 內容 |
|------|------|
| **文化資產類別** | 歷史建築 |
| **公告日期** | 2004年 (民國93年) |
| **主管機關** | 桃園市政府文化局 |
| **所在地** | 桃園市大溪區福安里 |

## 登錄理由

1. **歷史價值**: 見證大溪頭寮地區開發史
2. **建築價值**: 保存完整的清代閩南式建築
3. **家族史料**: 林家文書記錄地方開發歷程
4. **文化意義**: 反映清代漢人在隘墾區的生活樣態

## 林家文書

### 中央研究院臺灣史研究所收藏

「桃園大溪梅鶴山莊林家文書」已被中央研究院臺灣史研究所檔案館收藏並開放研究。

**文書內容涵蓋**:
- 土地契約
- 商業往來文件
- 家族譜系
- 歷史照片

**史料價值**:
- 記錄林本源家族與大溪開發的關係
- 反映清代至日治時期的社會經濟變遷
- 提供地方史研究的第一手資料

> 參考: [中央研究院臺灣史研究所檔案館 - 桃園大溪梅鶴山莊林家文書](https://archives.ith.sinica.edu.tw/news_con.php?no=363)

## 相關文化資產

### 三層地區社頭組織

梅鶴山莊所在的三層地區保有豐富的民俗文化:

| 社頭名稱 | 成立年份 | 主要特色 |
|---------|----------|----------|
| **樂豳社** | 1927年 | 北管、神轎，由梅鶴山莊族人創立 |
| **新勝社** | 1919年 | 龍陣代表 (1971年全國亞軍) |
| **福安社** | 1921年 | 福安宮所屬社頭 |

### 相關宗教建築

| 建築 | 關聯 |
|------|------|
| **齋明寺** | 1873年林登雲捐獻興建「齋明堂」，為齋明寺前身 |
| **福安宮** | 1815年建廟，1915年林六德捐獻土地遷建至現址 |
| **福仁宮** | 林登雲曾兩度擔任建醮爐主 (1884、1886年) |

## 保存與活化

### 修復工程

桃園升格後，文化部與桃園市政府合作推動修復:

- **第一期** (2016年完成): 正身修復
- **第二、三期** (2021年完成): 左右護龍修復

### 活化利用方向

1. **街角書屋**: 配合大溪木藝生態博物館概念
2. **古籍展示**: 展示山莊典藏的古籍、文書
3. **古蹟修復教育**: 提供古蹟修復類書籍供閱覽
4. **導覽服務**: 開放預約參觀與專業導覽

## 學術研究

### 相關研究資料

- 文化部文化資產局調查研究報告
- 大嵙崁文教基金會地方開發史料
- 國家文化記憶庫數位典藏

### 研究主題

1. 清代閩南建築研究
2. 大溪地區開發史
3. 林本源家族史
4. 隘墾區漢人聚落研究
5. 傳統建築修復技術
