# 梅鶴山莊參觀資訊 (Meihe Villa Visit Information)

## 基本資訊

| 項目 | 內容 |
|------|------|
| **名稱** | 梅鶴山莊 (Meihe Villa) |
| **地址** | 桃園市大溪區福安里頭寮一號 |
| **別稱地址** | 桃園市大溪區頭寮一路111號 |
| **文化資產類別** | 歷史建築 |
| **登錄年份** | 2004年 |
| **所屬轄區** | 桃園市政府文化局 |

## GPS 座標

- **緯度**: 約 24.847°N
- **經度**: 約 121.278°E

> 建議使用 Google Maps 搜尋「梅鶴山莊」或「桃園市大溪區頭寮一路111號」取得精確導航。

## 參觀方式

### 開放狀態
- 山莊**有條件開放**參觀
- **必須事先預約**，提供導覽服務

### 預約方式
- 須先聯繫林家族親會
- 建議透過大溪木藝生態博物館洽詢

### 開放空間
修復完成後，開放部分空間供參觀:
- 門樓
- 前庭
- 部分正身區域

## 未來規劃

1. **北橫國家風景區景點**: 未來有望成為北橫國家風景區計劃的重要景點
2. **街角書屋**: 配合大溪木藝生態博物館「街角書屋」概念活化經營
3. **閱覽服務**: 提供古蹟修復類書籍，以及山莊典藏的古籍、文書供民眾閱覽

## 交通方式

### 自行開車
- 國道三號大溪交流道下，往慈湖方向
- 沿台7線 (北橫公路) 行駛約10分鐘

### 大眾運輸
- 桃園客運: 搭乘往慈湖方向班車
- 於頭寮站下車步行約10分鐘

## 注意事項

1. **私人住宅**: 梅鶴山莊至今仍有林家後代居住，請尊重住戶隱私
2. **預約必要**: 未預約者可能無法入內參觀
3. **攝影限制**: 入內參觀請遵守管理規定
4. **活動資訊**: 特定節日 (如過年) 可能舉辦特展活動，可關注桃園市文化局公告
