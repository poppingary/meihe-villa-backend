# 梅鶴山莊歷史 (Meihe Villa History)

## 基本資訊

- **中文名稱**: 梅鶴山莊
- **英文名稱**: Meihe Villa / Meihe Mountain Manor
- **地址**: 桃園市大溪區福安里頭寮一號 (另一記載: 頭寮一路111號)
- **GPS座標**: 約 24.847°N, 121.278°E (大溪頭寮地區)
- **佔地面積**: 一甲
- **創建年份**: 1869年 (清同治8年)
- **文化資產登錄**: 2004年登錄為桃園市歷史建築

## 創建者

### 林登雲 (Lin Deng-yun, 1815-1888)

林登雲，字圻海，祖籍福建漳州。

**生平事蹟**:
- 1815年 (嘉慶20年): 出生
- 1829-1854年: 擔任林本源大嵙崁租館管事
- 1869年 (同治8年): 向林本源家族購買頭寮庄土地，興建「榮封第」(現梅鶴山莊)
- 1873年: 與江排呈、簡清雲捐獻興建「齋明堂」(現齋明寺基礎)
- 1875年 (光緒元年): 61歲時上奏清廷捐納同知，獲封五品大夫，在前落門楣上掛「榮封第」匾額
- 1884-1886年: 兩度擔任福仁宮建醮爐主
- 1888年: 逝世

林本源家族遷居板橋後，林登雲選擇在大溪定居，以「林本源商號總執事」為名號做生意。其後家業如雲，擁有礦場、米廠、磚廠、茶園、商號，成為大溪望族。

## 命名由來

1923年 (大正12年)，大溪文人**鄭永南**以北宋隱士**林和靖**「梅妻鶴子」典故，將原本的「榮封第」題名為「梅鶴山莊」。

> 林和靖 (林逋，967-1028) 為北宋著名詩人、隱士，一生不娶不仕，在杭州西湖孤山種梅養鶴，人稱「梅妻鶴子」。

## 林家開台史

| 年份 | 事件 |
|------|------|
| 1804年 | 林邦興 (第二十四世祖) 來台，為梅鶴林家開台祖 |
| 1809年 | 林邦興曾遷居宜蘭，後返霄裡定居 |
| 1815年 | 林登雲出生；同年福安宮建廟 |
| 1818年 | 林本源家族進入大嵙崁開發 |
| 1869年 | 林登雲建造「榮封第」|
| 1906年 | 林登雲三房之子林源松開發頭寮員潭坑煤礦 |
| 1927年 | 梅鶴山莊族人創立北管樂團樂豳社 |

## 頭寮地區開發

頭寮早期為隘墾區，與原住民常發生衝突。清末為防範泰雅族原住民，漢人在大溪三層最窄處設立第一座「隘勇寮」，此地因而得名「頭寮」。

梅鶴山莊選址面對泰雅族人進出的地點，具有挑戰性，需要家族團結合作。

**早期主要產業**:
- 香茅
- 茶葉
- 煤礦

**日治時期**: 設置製糖廠，周圍種植甘蔗

## 近代歷史

| 年份 | 事件 |
|------|------|
| 1955年 | 板橋林家捐獻慈湖土地 |
| 1962年 | 蔣介石將新埤更名「慈湖」|
| 1966年 | 板橋林家捐獻大溪陵寢土地 |
| 1967年 | 擴建頭寮大埤 |
| 1975年 | 蔣中正奉厝慈湖 |
| 1988年 | 蔣經國奉厝頭寮賓館，更名「大溪陵寢」|
| 2004年 | 梅鶴山莊登錄為歷史建築 |
| 2016年 | 第一期正身修復完成 |
| 2021年 | 第二、三期修復工程完工 |

戰後及興建慈湖陵寢時期，梅鶴山莊前庭及廂房曾有短暫駐軍。

## 林家現況

- 現已傳承七代，族人分布廣泛
- 2018年桃園市文化局立案成立「桃園市大溪林家梅鶴山莊族親會」

**著名族人**:
- 林熺達: 前大溪鎮長
- 朱林桂: 前桃園縣長朱立倫之母
